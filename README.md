## Features

-   Mondrian-style full-screen CSS grid
-   Scroll-driven text and color transitions within fixed panels
-   Simple portfolio teaser with links to my work (GitHub, p5.js)

## Tech Stack

-   HTML
-   CSS (CSS Grid, responsive layout)
-   Vanilla JavaScript

## Development

This is a static site. To run locally:

```bash
git clone https://github.com/<YOUR_USERNAME>/mondrian-animated-portfolio.git
cd mondrian-animated-portfolio
# Option 1: open index.html directly in a browser
# Option 2: serve with a simple local server (e.g. Live Server Plugin)
```
